This project is part of the Foundations course from The Odin Project. It focuses on building a basic recipe website using only HTML.

Skills Demonstrated
Basic HTML structure and elements:
<!DOCTYPE html>, <head>, <body>
Headings (<h1>–<h6>)
Paragraphs (<p>)
Links (<a href="">)
Images (<img src="">)
Cloning repositories from GitHub
Writing clear and concise commit messages
Understanding and applying the Git workflow

Purpose
The goal of this exercise is to practice structuring simple web pages while reinforcing fundamental Git and GitHub skills.
